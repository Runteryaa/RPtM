# 📦 ResourcePack to Mod Library (RPtMLib)

RPtM is a lightweight Fabric utility that lets you bundle multiple resource packs directly into a single mod JAR. It uses the `registerBuiltinResourcePack` API to register assets as "Built-in" packs. You can use [RPtM Converter](https://rptm.pages.dev/) to turn your resource packs to mods. All generated mods will be needing this library mod.

In Modrinth people tend to download mods over datapacks and resource packs. So i build that mod for people who suffer from that.

---

### 🛠️ Why use this?

* **Internal Loading:** Packaged assets stay inside the JAR—no more messy folders.
* **Automatic Detection:** If any RPtM mod is installed it will work without hustle.
* **Version Agnostic:** Built to work on **1.20+**.
* **Zero Bloat:** No heavy dependencies or performance hits.
* **Mod Compatibility:** Works with **[Respackopts](https://modrinth.com/mod/respackopts)** natively.
* **Open Source:** Both [RPtMLib](https://github.com/Runteryaa/RPtM) and [RPtM Converter](https://github.com/Runteryaa/RPtM-Converter) source code available on [GitHub](https://github.com/Runteryaa/RPtM).

### 🚀 Quick-Start
1. Go to [RPtM Converter](https://rptm.pages.dev/) website.
2. Upload your resource pack.
3. Fill in the blanks about your mod.
4. Click "Convert to Mod".
5. Drag-n-drop to your mods folder. (make sure you have RPtMLib installed)
6. DONE!
